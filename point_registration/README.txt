usage: main.py [-h] [--src SRC] [--tgt TGT]

Register two images by clicking on points in both images.

commands:
  ESC          Quit
  CLICK        Add point
  ENTER        Register the images
  BACKSPACE    Reset all points.

optional arguments:
  -h, --help  show this help message and exit
  --src SRC   source image, to be registered on the target image
  --tgt TGT   target image, onto which register the source image
